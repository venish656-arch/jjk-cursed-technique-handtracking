# JJK: Cursed Technique Hand Tracking 🔥

A real-time hand gesture controlled VFX experience inspired by Jujutsu Kaisen.

## ✨ Features

- Gesture detection using MediaPipe Hands
- 20,000 animated particles using Three.js
- Domain Expansion Effects:
  - 🔴 Reverse Cursed Technique: Red
  - 🟣 Hollow Purple
  - 🔵 Infinite Void
  - 🔥 Malevolent Shrine
- Real-time bloom and postprocessing
- Camera-based interaction

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Three.js
- MediaPipe Hands API
